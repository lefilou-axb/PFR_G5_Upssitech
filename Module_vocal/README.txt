Sur le nouveau PC, utiliser le terminal de commande pour aller dans 
le dossier cible, y placer les fichier Module_vocal.py et requirements.txt 
puis faire la commande "pip install -r requirements.txt"

Ne pas oublier de mettre en argument du main (int argc, char *argv[])
inclure la bibliotheque <stdlib.h>

mettre la commande "system("python Module_vocal.py");" dans le choix du menu pour lancer le programme python
